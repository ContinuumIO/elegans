# Elegans
A 3D plotting library for three.js and d3.js

![alt text](https://dl.dropboxusercontent.com/u/47978121/ss560.png)

## Supporting charts
* Surface
* Wireframe (not implemented yet)
* Scatter (not implemented yet)

## Usage
Please see examples.

## Dependency
Elegans depends on libraries below.
* D3.js
* three.js
* colorbrewer.js
